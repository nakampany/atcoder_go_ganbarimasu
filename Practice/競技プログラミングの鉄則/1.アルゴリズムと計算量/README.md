# ビット全探索

- bit 演算

```
x | y	x と y のビット単位 論理和
x ^ y	x と y のビット単位 排他的論理和
x & y	x と y のビット単位 論理積
x << n	x の n ビット左シフト
x >> n	x の n ビット右シフト
```
